 == Installation ==
Works with the Contact Form 7 plugin to send SMS notifications when somebody submits your contact form, using the API Configured By Site Admin 

1. You can:
 * Upload the `contact-form-sms-integration` folder to `/wp-content/plugins/` directory via FTP. 
 * Upload the full ZIP file via *Plugins -> Add New -> Upload* on your WordPress Administration Panel.
 * Search **Contact Form 7 SMS Integration** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate plugin through *Plugins* menu on WordPress Administration Panel.
3. Set up plugin on *Contact -> Sms Integration* or through *Settings* button.
4. Create Custome SMS Template *Contact -Contact Forms -> All Bulk SMS* and create your custome SMS Template.
5. Ready, now you can enjoy it, and if you like it and find it useful, make a [*donation*](https://www.allbulksms.in/payment.html).
